# HashMap Practice (Tasks 12-20)

В папке `map` добавлены новые упражнения по `HashMap`:

- **12_WordFrequencyCounter.java** — подсчет частоты слов в строке.
- **13_CharacterFrequency.java** — подсчет частоты символов и поиск самого частого.
- **14_GroupWordsByLength.java** — группировка слов по длине через `computeIfAbsent`.
- **15_FirstNonRepeatingCharacter.java** — поиск первого неповторяющегося символа.
- **16_TwoSumWithMap.java** — решение Two Sum через map (значение → индекс).
- **17_DetectDuplicatesWithMap.java** — вывод уникальных и дубликатов по частотам.
- **18_EqualsAndOrderInsensitive.java** — демонстрация `equals()` и `hashCode()` для map.
- **19_RemoveEntriesConditionally.java** — удаление записей по условию через `Iterator`.
- **20_MergeScores.java** — объединение карт с суммированием значений через `merge()`.
